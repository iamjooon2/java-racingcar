## 입력 기능

- [ ] 자동차 이름 입력 
  - [ ] 쉼표로 구분
  - [ ] 다섯 글자 이하인지 검증
  - [ ] 미입력 예외 처리
- [ ] 시도 횟수 입력
  - [ ] 미입력 예외 처리
  - [ ] 정수인지 검증
  - [ ] 양수인지 검증

- [ ] 입력 요구 메시지 출력

  - [ ] 시도 횟수
- [ ] 실행 결과 출력
